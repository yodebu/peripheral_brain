Tell iPython to load matplotlib and display all visuals created inline (that is, on this page)¶

In [25]:
%matplotlib inline

Import matplotlib's pyplot module¶

In [26]:
import matplotlib.pyplot as pyplot

Create a simple plot¶

In [27]:
pyplot.plot([1.6, 2.7])
